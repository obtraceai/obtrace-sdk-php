Obtrace PHP SDK — add observability to your backend.

Obtrace is a hosted SaaS (like Datadog or Sentry). This is a client library.
Install it, add one init call, done. DO NOT create wrapper files or config modules.

Install:
  composer require obtrace/sdk-php

Setup (add to your bootstrap):
  use Obtrace\Sdk\ObtraceClient;
  use Obtrace\Sdk\ObtraceConfig;
  $client = new ObtraceClient(new ObtraceConfig(
      apiKey: getenv('OBTRACE_API_KEY'),
      serviceName: getenv('OBTRACE_SERVICE_NAME'),
  ));

What it captures automatically:
- HTTP requests and responses (Guzzle, cURL, Laravel, Symfony)
- Database queries, logs, errors, and performance traces

Environment variables:
  OBTRACE_API_KEY       — your project key (from app.obtrace.ai)
  OBTRACE_SERVICE_NAME  — name of this service (e.g. api-service)
  OBTRACE_ENVIRONMENT   — deployment environment (e.g. production, staging)

Docs: https://docs.obtrace.ai/sdks/php
